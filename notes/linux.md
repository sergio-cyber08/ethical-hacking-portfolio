## 📂 Navigazione file system
```bash
ls      # lista file
cd      # cambia directory
pwd     # mostra percorso attuale
# 🐧 Appunti Linux
touch nomefile       # crea un file
mkdir cartella       # crea una cartella
rm nomefile          # elimina un file
rm -r cartella       # elimina una cartella
cp file1 file2       # copia file
mv file1 file2       # sposta/rinomina file
whoami            # mostra l'utente attuale
id                # mostra UID e gruppi
chmod 755 file    # cambia permessi
sudo comando      # esegui come admin
